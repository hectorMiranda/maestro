# Practice Routine — Db Minor Pentatonic

A short daily routine for the `db_minor_pentatonic` scale.

1. **Slow ascent** — play the scale up at 60 BPM, naming each note.
2. **Slow descent** — play it back down, listening for evenness.
3. **Up & down** — combine both hands, two octaves where comfortable.
4. **Arpeggio** — outline the tonic triad rooted on MIDI 61.
5. **Free play** — improvise four bars using only these notes.

Run the matching etude:

```sh
maestro play etude_db_minor_pentatonic_ascending
```
